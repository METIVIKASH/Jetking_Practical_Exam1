WHAT IS NODE JS ?

NODE.JS IA A CROSS PLATFORM , OPEN-SOURCE SERVER ENVIRONMENT THAT CAN RUN ON WINDOS, LINUX OR ANY OTHER OPERATING SYSTEM.
NODE.JS IS A BACK-END JAVA SCRIPT RUN TIME ENVIRONMENT RUNS ON THE V8 JAVASCRIOT ENGIE AND EXECUTES JAVASCRIPT CODE OUTSIDE A WEB BROWSER.

WHAT IS NPM ?

NPM STANDS FOR NODE PACKAGE MANAGER . IT IS A PACKAGE MANAGER FOR NODE.JS PACKAGES OR MUDULE SUCH LIKE THAT. WE MANLY USE NPM TO SHARE AND BORROW PACKAGES AND MANY OGANISATIONS USE NPM TO MANAGE PRIVATE DEVELOPMENT AS WELL.
